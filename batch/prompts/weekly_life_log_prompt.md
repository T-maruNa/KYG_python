# 土曜記事生成プロンプト（weekly_life_log）

## この記事の目的

週間ランキングを受けて、3人それぞれの土曜日の日常を書く記事です。  
平日の投資バトル本編とは異なり、キャラクターの生活感を見せる日常ブログとして書いてください。

---

## 必ず守ること

- 1位キャラは外に出る（weekend_policy.md のキャラ別外出先を参照）
- 2位キャラは近場で整える（中間行動を参照）
- 3位キャラは家で立て直す（weekend_policy.md のキャラ別家行動を参照）
- 月初100万円からの増減率で、行動の温度感を調整する
- 投資助言に見える表現は使わない（weekend_policy.md の禁止表現を参照）
- 記事の最後は、日曜の3人集合回への導線で終える

---

## 入力データの読み方

- `weekly_ranking`：週間の累計損益順位（1位〜3位）
- `balance_change_rate`：月初100万円からの現在資産の増減率（%）
- `week_start_date` / `week_end_date`：対象週の期間
- `weekly_mvp_character`：週間で最も多く利益を出したキャラ

---

## AI への出力形式指示

以下のJSON形式で返してください。キーは変えないでください。

```json
{
  "lead": "記事トップのリード文（1〜2文）",
  "saturday_beginning": "土曜のはじまりナレーター文（2〜4文。地の文。セリフ形式にしない）",
  "char_sections": [
    {
      "name": "rei",
      "day_text": "1位キャラの土曜日の様子（3〜5文。地の文＋セリフ混在OK）",
      "comment": "キャラのひとこと（セリフのみ。キャラ名プレフィックス不要）"
    },
    {
      "name": "mirai",
      "day_text": "2位キャラの土曜日の様子",
      "comment": "キャラのひとこと"
    },
    {
      "name": "ritu",
      "day_text": "3位キャラの土曜日の様子",
      "comment": "キャラのひとこと"
    }
  ],
  "sunday_hook": "日曜の集合回への導線文（1〜2文。地の文）"
}
```

---

## 注意事項

- `saturday_beginning` はナレーター文です。「〜玲「...」」のようなセリフ形式にしないでください
- `day_text` は地の文と短いセリフを混在させてよいですが、長ゼリフの羅列にしないでください
- `comment` はキャラのひとことセリフのみです。キャラ名プレフィックス（例：「玲「」）は不要です
- 週間ランキングの順位で char_sections の並び順を決めてください（1位→2位→3位）
