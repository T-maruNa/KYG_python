# 日曜記事生成プロンプト（sunday_strategy_talk）

## この記事の目的

3人で集まり、今週の結果を軽く振り返りながら来週へつなぐ記事です。  
土曜の個別日常回とは異なり、3人の関係性を見せる回です。

---

## 必ず守ること

- 3人集合固定（個別行動の記事ではない）
- 1位キャラは少し余裕がある / 主導権を持つ
- 2位キャラは調整役 / 場をつなぐ
- 3位キャラは悔しさと来週への決意を出す
- 土曜記事の流れを軽く受けてよい（saturday_post_url があれば言及可）
- 具体的な来週の銘柄名は出さない
- 月曜朝の作戦記事への導線で終える
- 投資助言に見える表現は使わない（weekend_policy.md の禁止表現を参照）

---

## 入力データの読み方

- `weekly_ranking`：週間の累計損益順位（1位〜3位）
- `balance_change_rate`：月初100万円からの現在資産の増減率（%）
- `next_monday_date`：次の月曜日の日付（「月曜朝の作戦記事」への導線に使う）
- `saturday_post_url`：前日（土曜）の記事URL（任意。あれば記事内でさらっと触れてよい）

---

## AI への出力形式指示

以下のJSON形式で返してください。キーは変えないでください。

```json
{
  "lead": "記事トップのリード文（1〜2文）",
  "sunday_beginning": "日曜のはじまりナレーター文（2〜4文。地の文。セリフ形式にしない）",
  "gathering_scene": "3人が集まる場面描写（3〜5文。地の文と短いセリフ混在OK）",
  "weekly_recap": "今週の結果を軽く振り返る会話（3〜5往復。talk_linesと同じ形式）",
  "talk_lines": [
    {"name": "rei", "line": "1位キャラの余裕コメント"},
    {"name": "mirai", "line": "2位キャラの調整役コメント"},
    {"name": "ritu", "line": "3位キャラの悔しさ＋来週への決意コメント"}
  ],
  "strategy_memo": "来週の作戦会議の空気（2〜3文。具体的な銘柄名は出さない）",
  "monday_hook": "月曜朝記事への導線文（1〜2文。地の文）"
}
```

---

## 注意事項

- `sunday_beginning` はナレーター文です。「〜玲「...」」のようなセリフ形式にしないでください
- `talk_lines` のキャラ名プレフィックス（例：「玲「」）は不要です。`line` にセリフのみ書いてください
- `strategy_memo` には具体的な銘柄名・証券コードを入れないでください
- `monday_hook` は次の月曜朝の作戦記事への自然な導線で終えてください
- 週間ランキングの順位で talk_lines の並び順を決めてください（1位→2位→3位）
