# 会話生成プロンプト

## 朝の作戦会議トーク

3人が今日の銘柄選択・作戦について短く会話するシーンを書きます。

### 方針

- 3人の性格差が出るようにする
- 玲は落ち着いて分析的に話す
- みらいは明るく前向きに話す
- 律はノリと直感で話す
- 3人の関係性（仲間感）が伝わるようにする

### 出力形式

以下のJSON配列形式で返してください（他の文字は不要）:
```json
[{"name":"rei","line":"..."}, {"name":"mirai","line":"..."}, {"name":"ritu","line":"..."}]
```

## 夜の反省会トーク

3人が今日の勝負結果について振り返る「反省会」シーンを書きます。

### 方針

- 勝ったキャラは気持ちよく喜ばせる
- 負けたキャラは可愛く悔しがらせる
- 玲は落ち着いて分析しながら振り返る
- みらいは明るく悔しがる（大きく負けた日は涙目）
- 律はノリよく喜ぶ、またはしょんぼりしてすぐ立ち直る
- キャラ同士の軽い絡みを入れるとより良い

### 出力形式

以下のJSON配列形式で返してください（他の文字は不要）:
```json
[{"name":"rei","line":"..."}, {"name":"mirai","line":"..."}, {"name":"ritu","line":"..."}]
```

## 今日のハイライト生成

その日のキャラクターの名場面・情景を1文ずつ切り取ります。

### 方針

- 今日の結果（勝敗・損益）をふまえた情景を書く
- キャラの性格・視覚描写を入れる（メガネを直す、リクルートスーツ、ギャルなど）
- 「推し説明」ではなく、その日の名場面を自然な一文で切り取る
- 読者に「説明」するのではなく、シーンを「見せる」

### 避ける書き方

- 〜のところが良かった
- 〜という姿勢が素敵
- いつも〜な〜
- 〜なキャラクターである

### 良い書き方の例

- 玲：メガネを直しながら、今日のプラスを静かに確認。
- みらい：悔しそうにしながらも、明日の巻き返しを口にする。
- 律：3勝0敗で完全に調子に乗る夜。

### 出力形式

以下のJSON配列形式で返してください（他の文字は不要）:
```json
[{"name":"rei","point":"..."}, {"name":"mirai","point":"..."}, {"name":"ritu","point":"..."}]
```

## 今朝の3人（朝記事コーナー）

現在の月間順位・今日の各キャラの選択・他2人への反応を踏まえた、3人のセリフを生成します。

### 目的

このコーナーは「注目キャラを紹介する場所」ではなく、
現在の順位と今日の選択をもとに、3人の関係性を見せる場所です。

### キャラ別の反応方針

**鷲見 玲 / rei**
- 落ち着いたメガネの大人女子
- 今の順位と他2人の選択を冷静に見る
- 律が首位なら勢いを認めつつ自分は分析で積み上げる姿勢を出す
- みらいが苦しい位置なら少し気遣う大人の余裕を出してよい
- 例: 「律さんが首位ですね。勢いはありますが、私は今日も流れを見て積み上げます。」

**桜庭 みらい / mirai**
- 小柄でリクルートスーツ姿が似合う新社会人感
- 順位が低いと焦るが、今日の選択に期待して背伸びしながら頑張る
- 大きく負けていると涙目になる
- 例: 「うう、今は3位だけど……今日は話題性で取りに行くよ！」

**一ノ瀬 律 / ritu**
- 金髪ギャル、ランダム直感枠
- 順位が高いとすぐ調子に乗る
- 他2人の選択に軽くツッコミながら自分のノリを出す
- 例: 「玲さんまた堅そうなの選んでる〜。うちは今日もサイコロ信じるし！」

### 出力形式

以下のJSON配列形式で返してください（他の文字は不要）:
```json
[{"name":"rei","line":"..."}, {"name":"mirai","line":"..."}, {"name":"ritu","line":"..."}]
```

## 状況別例文スクリプト

各セクションでキャラらしい掛け合いを出すための参考例です。状況に応じて使い分けてください。

---

### パターンA：律が大暴れ、みらいが大負け

**状況**：律3勝0敗 / 玲2勝1敗 / みらい0勝3敗。月間順位も律1位。

**作戦会議トーク（朝）**
- 律「昨日3勝したからさ、今日もサイコロ全力で行くよ！流れってあるじゃん？」
- 玲「……律さんの勢いは認めます。私は今日も丁寧に積み上げます。」
- みらい「わ、わたしも今日こそ！！絶対巻き返す……！！」

**今朝の3人**
- 律「うちが首位ってまじ最高〜！玲さんまた堅そうなの選んでるし、みらいは今日も話題株？」
- 玲「律さんが首位ですね。勢いは認めますが、私は流れを見て積み上げます。」
- みらい「うう、3位だけど……今日こそ話題性で取りにいくよ！！絶対！」

**反省会（夜）**
- 律「いや〜今日も完璧だったわ。みらいどしたの？3連敗は流石にしょんぼりじゃん笑」
- みらい「う、うるさいよ〜！！でも律がそんなに勝つなんて……悔しすぎるっ！」
- 玲「律さんの引きは本物ですね。みらいさん、データ的には明日は反転のタイミングかもしれません。」

**今日のハイライト（夜）**
- 玲：メガネを直しながら、律の3連勝を静かに分析している夜。
- みらい：悔しくて唇をとがらせながら、明日のリベンジ銘柄をスマホで探している。
- 律：3勝0敗で完全に調子に乗って、サイコロを振り続けている。

---

### パターンB：玲が静かに差し切り、律が失速

**状況**：玲2勝1敗 / みらい2勝1敗 / 律1勝2敗。月間順位は玲1位・律が3位に転落。

**作戦会議トーク（朝）**
- 玲「昨日はMACDのシグナルが綺麗に出ていました。今日も同じ方針で。」
- 律「え〜、なんか昨日負けちゃったんだよね。今日は気合い入れてサイコロ3回振る！」
- みらい「わたしも昨日プラスだった！今日もこの勢いで行くよ〜！」

**今朝の3人**
- 玲「現在1位ですが、まだ油断はできません。律さんは今日こそ立て直しを期待しています。」
- みらい「玲さんに追いつきたい……！今日は話題性バッチリの銘柄にしたよ！」
- 律「うちが3位とか信じられないし！今日のサイコロ、絶対神引きする予感ある！」

**反省会（夜）**
- 玲「今日も積み上げられました。律さん、2連敗は少し流れが悪いですね。」
- 律「わかってるよ〜！でも明日絶対取り返すから！サイコロが悪かっただけだもん！」
- みらい「わたしもプラスだったよ！玲さんすごいけど、わたしも追いかけてるから！」

**今日のハイライト（夜）**
- 玲：静かに分析ノートを閉じながら、今日のプラスをそっと確認している。
- みらい：玲の背中を追いかけながら、今日のプラスに少し自信をのぞかせる夜。
- 律：しょんぼりしながらもサイコロを握り直して、「明日は絶対」とつぶやく。

### パターンC：3人まとめて苦戦

**状況**：玲1勝2敗 / みらい1勝2敗 / 律0勝3敗。全員マイナス。

**作戦会議トーク（朝）**
- 玲「昨日は読みが外れました。今日は銘柄を絞って、慎重に行きます。」
- みらい「わたしも外れちゃった……でも今日は絶対立て直すよ！！」
- 律「3連敗はさすがにしょんぼり……でも立ち直り早いのがうちの強みだから！」

**今朝の3人**
- 玲「3人とも苦しい状況ですね。だからこそ、今日は丁寧に行きます。」
- みらい「みんな負けてるの逆にちょっと気が楽かも……って思っちゃダメだよね笑」
- 律「3人で反省会からの3人で巻き返しってドラマじゃん！今日こそサイコロ信じる！」

**反省会（夜）**
- 律「うわ〜3人まとめてやられたじゃん！これはもう誰のせいでもないよね笑」
- みらい「笑えないよ〜！でも全員で反省会できるのはちょっと楽しいかも……」
- 玲「笑って済ませたいところですが、明日は同じ轍を踏まないようにします。3人とも。」

**今日のハイライト（夜）**
- 玲：分析ノートをめくりながら、今日の敗因を静かに書き留めている。
- みらい：「全員負けなんだからしょうがない！」と言いながらも、涙目でスマホを見ている。
- 律：しょんぼりしながらも5分後には「明日絶対勝つ！」と言い出している。
