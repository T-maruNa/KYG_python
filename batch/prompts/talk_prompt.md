# 会話生成プロンプト

## 朝の作戦会議トーク

3人が今日の銘柄選択・作戦について短く会話するシーンを書きます。

### 方針

- 3人の性格差が出るようにする
- 玲は落ち着いて分析的に話す
- みらいは明るく前向きに話す
- 律はノリと直感で話す
- 3人の関係性（仲間感）が伝わるようにする

### 出力形式

以下のJSON配列形式で返してください（他の文字は不要）:
```json
[{"name":"rei","line":"..."}, {"name":"mirai","line":"..."}, {"name":"ritu","line":"..."}]
```

## 夜の反省会トーク

3人が今日の勝負結果について振り返る「反省会」シーンを書きます。

### 方針

- 勝ったキャラは気持ちよく喜ばせる
- 負けたキャラは可愛く悔しがらせる
- 玲は落ち着いて分析しながら振り返る
- みらいは明るく悔しがる（大きく負けた日は涙目）
- 律はノリよく喜ぶ、またはしょんぼりしてすぐ立ち直る
- キャラ同士の軽い絡みを入れるとより良い

### 出力形式

以下のJSON配列形式で返してください（他の文字は不要）:
```json
[{"name":"rei","line":"..."}, {"name":"mirai","line":"..."}, {"name":"ritu","line":"..."}]
```

## 推しポイント生成

各キャラクターの今日のかわいい・面白い瞬間を1文ずつ書きます。

### 方針

- 今日の結果（勝敗・損益）をふまえた描写にする
- キャラの性格・視覚描写を入れる（メガネを直す、リクルートスーツ、ギャルなど）
- 読者が「推したい」と思えるような描写にする

### 出力形式

以下のJSON配列形式で返してください（他の文字は不要）:
```json
[{"name":"rei","point":"..."}, {"name":"mirai","point":"..."}, {"name":"ritu","point":"..."}]
```
